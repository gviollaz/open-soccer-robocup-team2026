# Journal de Ingeniería

Este directorio contiene el **diario cronológico** del proyecto. Cada entrada documenta trabajo realizado, decisiones tomadas, problemas encontrados y aprendizajes.

## Formato de entradas

Nombre de archivo: `YYYY-MM-DD-descripcion-breve.md`

Cada entrada debe tener:

```yaml
---
title: "Descripción de la sesión"
date: 2026-MM-DD
author: "Nombre"
ai-assisted: false
tags: [area1, area2]
---
```

## Contenido sugerido por entrada

- **Objetivo**: Qué se quería lograr
- **Trabajo realizado**: Qué se hizo
- **Resultados**: Qué funcionó, qué no
- **Problemas encontrados**: Obstáculos y cómo se resolvieron (o no)
- **Próximos pasos**: Qué queda pendiente
- **Fotos/Videos**: Links si aplica
