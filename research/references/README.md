# Proyectos y recursos de referencia

Proyectos, papers y recursos externos para estudiar y aprender.

## Equipos de referencia

### ducc (Singapore, Hwa Chong Robotics Club)
- **Repo**: https://github.com/8076ducc/robocup-2025
- Equipo de 6 miembros que compite en Soccer Open y Lightweight
- Código y documentación completa publicada
- Referencia valiosa para nivel de competencia internacional

## Recursos oficiales de RoboCupJunior

### Team Description Papers (TDPs)
- **TDPs 2025**: https://github.com/robocup-junior/rcj-soccer-tdp-2025
- Papers de todos los equipos participantes en el mundial
- Fuente primaria para entender técnicas de equipos top

### Awesome RCJ Soccer
- **Lista curada**: https://github.com/robocup-junior/awesome-rcj-soccer
- Websites, documentos, posters, videos de todas las ediciones
- TDPs históricos desde 2021

### Communication Module
- **Repo**: https://github.com/robocup-junior/soccer-communication-module
- Esquemáticos, código fuente, documentación del módulo obligatorio
- Interfaz GPIO + alimentación

### Pelota IR Open Source (2026)
- **Repo**: https://github.com/robocup-junior/ir-golf-ball
- Nueva pelota de 42mm para Soccer Infrared
- Open source: archivos e instrucciones de fabricación

### Simulador de Soccer
- **Repo**: https://github.com/robocup-junior/rcj-soccersim
- Simulador basado en Webots
- Útil para probar estrategias sin hardware físico

### Robot Soccer Kit (open source)
- **Repo**: https://github.com/robot-soccer-kit
- Kit completo open source para RCJ Soccer
